# Pick 6 Admin

Dashboard and analytics.

### Version
1.0.0


### Installation

Please make sure you have [node](http://nodejs.org/) installed.

You need Gulp and Bower installed globally:

```sh
$ npm install -g gulp bower
```

[Gulp Angular Generator](https://github.com/Swiip/generator-gulp-angular/) was used to set up the app.

Run the following commands to install dependencies:
```sh
$ npm install
$ bower install
```

Run the following command to run app locally:

```sh
$ gulp serve
```

To build for production please run:
```sh
$ gulp build
```
